# Виджет банковских операций клиента

## Данный виджет предназначен для банковских операций клиента:
1. Маскировка номера карты клиента;
2. Маскировка номера счета клиента;
3. Форматирование даты и представление в виде "ДД.ММ.ГГГГ".

### 1. Установка:

1. ***Клонируйте репозиторий по ссылке HTTPS или SSH:***

   Ссылка HTTPS: ```https://github.com/AzatAlfredovich/bank_card_widget.git```

   Ссылка SSH: ```git@github.com:AzatAlfredovich/bank_card_widget.git```

2. ***Перейдите в директорию проекта:***

   C:\Users\User\PycharmProjects\bank_card_widget

3. ***Установите необходимые зависимости через консоль:***

   pip install -r requirements.txt

### 2. Примеры функций:

   - 1.Функция, принимающая на вход номер карты и возвращающая маску номера, не показывая часть, но показывая 4 последние цифры
```
Вводим: print(get_mask_card_number("7000 7922 8960 6361"))
Получаем: 7000 79** **** 6361
```
  - 2.Функция, принимающая на вход номер аккаунта и возвращающая маску номера, 4 последние цифры, перед которыми **
```
Вводим: print(get_mask_account("7000792289606361"))
Получаем: **6361
```
  - 3.Функция, принимающая на вход номер карты или счета, возвращающая маску с частичным номером карты или последними 4 цифрами номера счета
```
Вводим: print(mask_account_card("Visa Gold 5999414228426353"))
Получаем: Visa Gold 5999 41** **** 6353
```
  - 4.Функция, принимающая на вход строку в формате, возвращающая чистую дату
```
Вводим: print(get_date("2025-05-23T02:26:18.671407"))
Получаем: 23.05.2025
```
  - 5.Функция, фильтрующая транзакции по заданной валюте и возвращающая итератор
```
Вводим: список словарей с нужным ключом "currency"
Получаем: список словарей только с нужной валютой при наличии
```
  - 6.Генератор, поочередно выдающий описание транзакции
```
Вводим: список словарей с транзакциями
Получаем: описание операций при наличии - что, куда и откуда отправлено
```
  - 7.Генератор номеров карт в заданном диапазоне
```
Вводим: диапазон с количеством необходимых карт
Получаем: ряд номеров карт по порядку в зависимости от ввода
```

### 4. Тестирование 
Тестовые данные приведены в пакете "tests", который включает в себя модули с проверками и фикстурами, с HTML-отчетом о покрытии тестами вы можете ознакомиться в репозитории
```
conftest.py - файл с фикстурами
test_masks.py, test_processing.py, test_widget.py, test_decorators.py, test_generators.py - файлы с тестами функций
```
Для ознакомления с HTML-отчетом введите в консоль следующие команды:

Вариант 1. При активированном виртуальном окружении:
```
pytest --cov
```
Вариант 2.  Через poetry:
```
poetry run pytest --cov
```
Вариант 3. Чтобы сгенерировать отчет о покрытии в HTML-формате, где src — пакет c модулями, которые тестируем:
```
pytest --cov=src --cov-report=html
```
